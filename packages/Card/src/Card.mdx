# Card Usage

```tsx
<div style={{ display: 'grid', gridTemplateColumns: '1fr 1fr', gridGap: 24, padding: '16px 8px' }}>
    <Card>
        This is a card to place content inside of
    </Card>
    <Card as="section">
        This card is rendered as a section as opposed to a div
    </Card>
</div>
```
