# Checkbox Usage

## Basic Usage

All checkboxes are controlled elements, meaning you must pass the "checked" prop and "onChange" prop.

```tsx
<div style={{ display: 'grid', gridTemplateColumns: '100%', gridGap: 16 }}>
    <div>
        <Checkbox /> <Checkbox disabled /> Unchecked
    </div>
    <div>
        <Checkbox checked={true} /> <Checkbox disabled checked={true} /> Checked
    </div>
    <div>
        <Checkbox checked={'mixed'} /> <Checkbox disabled checked={'mixed'} /> Intermediate
    </div>
    <div>
        <Checkbox checked={false} invalid /> <Checkbox checked={true} invalid /> <Checkbox checked={'mixed'} invalid /> Invalid states <br />
    </div>
</div>
```

## Implicit vs Explicit Checkbox Labels

To make checkboxes <a href="https://www.w3.org/TR/html401/interact/forms.html#edef-LABEL" target="_blank" rel="noopener noreferrer nofollow">accessible</a>, you must use one of the two label associations: **explicit** or **implicit**. 

**Explicit labels** connect to the input using the htmlFor prop, while **implicit labels** wrap around the input. These associations allow you to toggle the checkbox by clicking the label text.

```tsx
<>
    <div style={{ marginBottom: '16px' }}>

        {/* Explicit association with htmlFor and the input's id */}
        <Checkbox id="emailOptIn" aria-labelledby="emailOptInLabel" />
        <label htmlFor="emailOptIn" id="emailOptInLabel">
            {" "}Send me updates on this product
        </label>

    </div>
    <div>

        {/* Implicit association by wrapping the label tag around the input */}
        <label>
            <Checkbox /> I agree to the terms and conditions
        </label>

    </div>
</>
```
