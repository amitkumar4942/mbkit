# Dialog Usage

## Basic Usage

When the "show" prop is true, the contents of the modal will be read to a screen reader. 

Using the "leastDestructiveRef" property will set the focus to the ref passed in when the modal opens. Use this on buttons like "Cancel" and never for destructive actions like deleting something. By default, the Close button will receive focus.

```tsx
<Component initialState={{ show: false }}>
    {({ state, setState }) => (
        <>
            <button onClick={() => setState({ show: true })}>Show Dialog</button>
            <Dialog header="Generic Dialog" show={state.show} onClose={() => setState({ show: false })}>
                <p>This is a generic dialog that should be used to get a response from a user</p>
                <div style={{ display: 'flex', justifyContent: 'flex-end' }}>
                    <button onClick={() => setState({ show: false })}>Cancel</button>
                    <button onClick={() => setState({ show: false })}>Save</button>
                </div>
            </Dialog>
        </>
    )}
</Component>
```
