# Typography Usage

## Text Component

By default, the Text component is a paragraph tag, but the "size" prop allows you to change the size. 

When you want to use a heading (h1, h2, h3, etc.), consider using the "Heading" component instead.

```tsx
<>  
    <Text size={1}>Size 1</Text>
    <Text size={2}>Size 2</Text>
    <Text size={3} bold as="span">Size 3</Text>
    <Text size={4}>Size 4</Text>
    <Text size={5}>Size 5</Text>
    <Text size={6}>Size 6</Text>
    <Text size={'body'}>Size 7 can be a "sub-heading" for an h1.</Text>
    <Text>Size 'body' is the default paragraph size and default "size" prop</Text>
    <Text size={'body-small'}>Size 'body-small' is used for meta text, Tipsies, or Quicknotes. This can be a placeholder or error text. (If you show this conditionally, don't forget to add your aria labels!)</Text>
    <Text size={'pre-heading'}>Size 'pre-heading' is used for displaying secondary headings with large font as body text.</Text>
</>
```

## Render "as"

You can render the Text component as another HTML tag using the "as" prop.

```tsx
<Text as="label" size={9} color="meta">This is a label now 
  <br />
  <input defaultValue="Edit this text" />
</Text>
```

## Colored Text

Apply branded colors to your text with the "color" prop.

```tsx
<>
    <Text>This is the default color</Text>
    <Text color="primary">This is the "primary" brand color</Text>
    <Text color="info">This is the info or "secondary" brand color</Text>
    <Text color="success">This is the "success" color</Text>
    <Text color="warning">This is the "warning" color</Text>
    <Text color="error">This is the "error" color</Text>
    <Text color="meta">This is the "meta" color</Text>
</>
```

## Heading Component

Use the "as" prop to specify the semantic heading tag.

```tsx
<>
    <Heading as="h1">This is an h1</Heading>
    <Heading as="h2">This is an h2</Heading>
    <Heading as="h3">This is an h3</Heading>
    <Heading as="h4">This is an h4</Heading>
    <Heading as="h5">This is an h5</Heading>
    <Heading as="h6">This is an h6</Heading>
</>
```

## Custom Heading Sizing

Need h1 sizing but want to render h6? Try using the "size" prop.

```tsx
<>
    <Heading as="h6" size={"hero"}>I'm a big h6</Heading>
    <Heading as="h1" size={6}>I'm a small h1</Heading>
</>
```

## Colored Heading

Apply branded colors to your heading with the "color" prop.

```tsx
<>
    <Heading as="h2">This is the default color</Heading>
    <Heading as="h2" color="primary">This is the "primary" brand color</Heading>
    <Heading as="h2" color="info">This is the "info" color</Heading>
    <Heading as="h2" color="success">This is the "success" color</Heading>
    <Heading as="h2" color="warning">This is the "warning" color</Heading>
    <Heading as="h2" color="error">This is the "error" color</Heading>
    <Heading as="h2" color="meta">This is the "meta" color</Heading>
</>
```
