# ErrorMessage Usage

## Basic Usage

```tsx
<>
    <input type="email" style={{ display: 'block', width: '100%', padding: 8, margin: '8px 0', }} />
    <ErrorMessage show={true}>Please enter a valid email address</ErrorMessage>
</>
```

## Interactive Email Demo

Turn on your screen reader of choice, then fill out this field with an invalid email address and tab out. The screen reader will read the error message.

## Polite vs Assertive

ErrorMessage uses <a href="https://ui.reach.tech/alert/" target="_blank" rel="noopener noreferrer nofollow">Reach UI's Alert package</a>, which allows you to interrupt the screen reader. 

You generally want to use the default "polite" setting, which allows the screen reader to finish reading. The "assertive" setting interrupts the screen reader and should be used sparingly. 

```tsx
<>
    <input type="email" style={{ display: 'block', width: '100%', padding: 8, margin: '8px 0', }} />
    <ErrorMessage show={true} type="assertive">Please enter a valid email address</ErrorMessage>
</>
```
