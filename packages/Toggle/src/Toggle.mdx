# Toggle Usage

## Basic Usage

Under the hood, Toggle is a checkbox. 

```tsx
<>
    <Toggle checked={false} onChange={() => {}} />
    <Toggle checked={false} onChange={() => {}} size={2} />

    <br />
    
    <Toggle checked onChange={() => {}} />
    <Toggle checked onChange={() => {}} size={2} />
</>
```

## Disabled State

```tsx
<>
    <Toggle disabled onChange={() => {}} />
    <Toggle disabled onChange={() => {}} size={2} />

    <br />
    
    <Toggle disabled checked onChange={() => {}} />
    <Toggle disabled checked onChange={() => {}} size={2} />
</>
```

## Label with Toggle

```tsx
<>
    <Toggle id="feature-a" aria-labelledby="feature-a-label" />{" "}
    <label htmlFor="feature-a" id="feature-a-label">Feature A</label>
</>
```
