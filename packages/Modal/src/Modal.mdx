# Dev Documentation

By default modals are fluid and will respond based on the width of the window up to a max-width of 960 pixels which is the size 1 default. Size 2 is the medium size and size 3 is the smallest. Any customization beyond this should be added via a CSS class name. 

<RenderOnly>
    <ModalExample buttonText="Modal size 1 (960px max width)" size={1} />
    <ModalExample buttonText="Modal size 2 (670px max width)" size={2} />
    <ModalExample buttonText="Modal size 3 (480px max width)" size={3} />
</RenderOnly>

Below are live examples of the modal with different props. The content section contains no inner styling relating to padding so you can use your own grid/padding or custom background

<RenderOnly>
    <ModalExample size={1} buttonText="Open Basic Modal (Few children)" />
    <ModalExample size={1} buttonText="Open Basic Modal (Many children)" renderManyChildren />
</RenderOnly>

The header/footer both accomidate the correct styling since there has been little deviation historically. 

<RenderOnly>
    <ModalExample size={1} buttonText="Open Modal with title (Few children)" header="Hello world" />
    <ModalExample size={1} buttonText="Open Modal with title (Many children)" renderManyChildren header="Hello MINDBODY" />
    <ModalExample 
        size={1}
        buttonText="Open Modal with title and footer (Few children)"

    />

</RenderOnly>
