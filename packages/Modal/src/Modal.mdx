# Modal Usage

## Basic Usage

By default, modals are fluid. Their max-width is determined by the "size" prop. Size 1 is 960px, size 2 is 670px, and size 3 is 480px. Any customization beyond this should done with a custom CSS class. 

<RenderOnly>
    <ModalExample buttonText="Modal size 1 (960px max width)" size={1} />
    <ModalExample buttonText="Modal size 2 (670px max width)" size={2} />
    <ModalExample buttonText="Modal size 3 (480px max width)" size={3} />
</RenderOnly>

## Modal with Inner Styles

Modals contain no inner styling, so you can use your own grid, padding, or custom background.

<RenderOnly>
    <ModalExample size={1} buttonText="Open Basic Modal (Few children)" />
    <ModalExample size={1} buttonText="Open Basic Modal (Many children)" renderManyChildren />
</RenderOnly>

## Modal Header and Footer

The "header" and "footer" props provide a pre-styled header and footer, respectively.

<RenderOnly>
    <ModalExample size={1} buttonText="Open Modal with title (Few children)" header="Hello world" />
    <ModalExample size={1} buttonText="Open Modal with title (Many children)" renderManyChildren header="Hello MINDBODY" />
    <ModalExample 
        size={1}
        buttonText="Open Modal with title and footer (Few children)" 
        header="Hello world" 
        footer={<button>This button does nothing</button>}
    />
    <ModalExample 
        size={1}
        buttonText="Open Modal with title and footer (Many children)" 
        header="Hello world" 
        renderManyChildren
        footer={<button>This button does nothing</button>}
    />
</RenderOnly>
