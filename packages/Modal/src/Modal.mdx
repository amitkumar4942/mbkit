By default modals are fluid and will respond based on the width of the window up to a max-width of 960 pixels which is the size 1 default. Size 2 is the medium size and size 3 is the smallest. Any customization beyond this should be added via a CSS class name. 

<RenderOnly>
    <ModalExample buttonText="Modal size 1 (default 960px max width)" size={1} />
    <ModalExample buttonText="Modal size 2 (670px max width)" size={2} />
    <ModalExample buttonText="Modal size 3 (480px max width)" size={3} />
</RenderOnly>
