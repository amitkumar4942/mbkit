# FileUploader Usage

## Basic Usage

FileUploader supports single file uploads by default. 

*Note: Please use labels with FileUploader for proper accessibility. For demo purposes, only the first example includes a label.*

```tsx
<>
    <label htmlFor="uploadFile" id="uploadFileLabel">
        Upload an image
    </label>
    <FileUploader 
        id="uploadFile"
        aria-labelledby="uploadFileLabel"
        value={null} 
        onChange={(event) => console.log(event.target.files)} 
        label="Upload your files"
    />
</>
```

## Multiple Files

FileUploader also supports multiple file uploads using the "multiple" prop.

```tsx
<FileUploader 
    multiple 
    value={null} 
    onChange={(event) => console.log(event.target.files)} 
/>
```

## Single File as Value

```tsx
<FileUploader 
    value={'some-file.jpg'} 
    onChange={(event) => console.log(event.target.files)} 
/>
```

## Multiple Files as Value

```tsx
<FileUploader 
    multiple 
    value={['some-file.jpg', 'another-file.json', 'test.csv']} 
    onChange={(event) => console.log(event.target.files)} 
/>
```
