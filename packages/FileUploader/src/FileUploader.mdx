# Dev Documentation

Basic use case of single file uploader. Use a label for accessibility, examples after the first do not include a label for brevity. 

```tsx
<>
    <label htmlFor="uploadFile" id="uploadFileLabel">
        Upload an image
    </label>
    <FileUploader 
        id="uploadFile"
        aria-labelledby="uploadFileLabel"
        value={null} 
        onChange={(event) => console.log(event.target.files)} 
    />
</>
```

Basic use case of multiple file uploader

```tsx
<FileUploader 
    multiple 
    value={null} 
    onChange={(event) => console.log(event.target.files)} 
/>
```

Using a single file as a value

```tsx
<FileUploader 
    value={'some-file.jpg'} 
    onChange={(event) => console.log(event.target.files)} 
/>
```

Multiple file value

```tsx
<FileUploader 
    multiple 
    value={['some-file.jpg', 'another-file.json', 'test.csv']} 
    onChange={(event) => console.log(event.target.files)} 
/>
```
