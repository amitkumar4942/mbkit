# Label Usage

## Basic Usage

Label components should be used with input fields and select dropdowns. 

```tsx
<>
    <Label 
        htmlFor="name" 
        id="name-label"
    >
        Enter your name
    </Label>
    <input 
        id="name" 
        aria-desribedby="name-label, name-helper-text" 
        style={{ display: 'block', width: '100%', padding: 8, margin: '4px 0' }} 
    />
    <Label 
        htmlFor="name" 
        id="name-helper-text"
    >
        Use your real name
    </Label>
</>
```

## Using the Default Label Tag

Use the default browser label for checkboxes; this text should be the same size as standard body copy.

```tsx
<>
    <input type="checkbox" id="terms" aria-labelledby="terms-label" />{" "}
    <label htmlFor="terms" id="terms-label">I agree to the terms and conditions</label>
</>
```
