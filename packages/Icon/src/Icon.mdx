## Targeting icons with SCSS 

This snippet targets all path, g, and use elements where they have a fill prop that contains a pound sign which indicates that it is using a hex code. 

<CodePreview language="scss">
    svg g:not([fill~="#"]),
    svg path:not([fill~="#"]),
    svg use:not([fill~="#"]) { 
        fill: #ffffff;
    }
</CodePreview>
