# Tipsy Usage

_Note: window collision detection of both the left and right will take precedence to any set position_

```tsx
<TipsyExamples multiLineExample={true} />
```

Tipsy position aligned with "top-left" of trigger element (default)

```tsx
<div style={{
    display: 'grid',
    gridTemplateColumns: `50% 50%`,
    gridGap: '8px',
    margin: '1em 0',
    position: 'relative'
}}>
    <div>
        <Tipsy label="Don't put anything critical to getting the task at hand complete in here"><button>Hover Me</button></Tipsy>
    </div>
    <div>
        <Tipsy label="These are useful for tips or helper text"><button>Hover Me</button></Tipsy>
    </div>
    <div>
        <Tipsy label="Assist the user with these messages"><button>Hover Me</button></Tipsy>
    </div>
    <div>
        <Tipsy 
            label={
                <>
                    All doggies go to heaven (or so I've been told).<br />
                    They run and play along the streets of Gold.<br />
                    Why is heaven such a doggie-delight?<br />
                    Why, because there's not a single cat in sight...
                </>
            }
        >
            <button style={{ float: 'right' }}>Multiline tipsy</button>
        </Tipsy>
    </div>
</div>
```
